大会开场演讲和日程介绍
10:00-10:10
django REST 快速开发
10:10-10:55 张海波
基于django restful 框架做快速 API开发, 大环境,微服务盛行的情况下 API 重要, 快速构建 API 的方法能极大提高企业生产力,快速应对需求和迭代变化, 主要讲解下 如何 设计 restful 以及如何快速开发一个 restful 的 api... 以及一些和 API 相关的衍生
Python在主机白盒安全检测中的应用
10:55-11:40 李毅为
安全领域有非常多优秀的 Python 工具和框架。本议题介绍主机白盒安全检测的概念和阿里巴巴在海量服务器的场景下对主机白盒安全检测的需求。深度介绍如何使用 Python 、一系列 Python 开源框架和 MongoDB 完成阿里巴巴主机白盒安全检测系统，为安全领域上层丰富的业务场景提供技术支持。
Graphene让Web应用迭代加速
11:40-11:50 麦伟祺
随着需求变化愈发的激烈，对迭代速度的要求也越来越高。Graphene是GraphQL的Python实现，能帮助web开发者开发出能持续快速迭代的应用。演讲会介绍GraphQL的基本原理，包括理念和基本定义；GraphQL与REST的对比；Graphene的基本用法，并用demo展示如何让Web应用的迭代速度加快。

下午

MatrixData在气象数据开放大赛的获奖作品分享
13:00-13:45 邓静芝
国信优易数据杯2016全国开放大数据创新创意大赛，暨国家气象信息中心首届气象数据开放大赛，MatrixData团队经过长达3个月的初赛、复赛，最终在总决赛中获奖。队长dengjingzhiData就此次参赛做主题分享，介绍参赛历程，详细解读MatrixData获奖项目《气象数据在药品供应链优化中的应用》，内容涉及Pandas数据处理、Scikit-learn机器学习等Python工具链的运用。
Head First SaltStack
13:45-14:30 竺夏栋
主题介绍了作者是如何在短时间内利用Python的一些运行时特性作为分析手段迅速掌握SaltStack这款自动化运维框架背后的运行原理与架构。希望通过本主题，与会嘉宾不仅能够了解到SaltStack的体系架构设计，更重要的，是如何去剖析一款框架的方法。因此，本主题会被设计成非运维开发领域的Python爱好者也感兴趣、能有所收获的大纲形式。
Python技术创业小记
14:30-15:15 钟锦威
作者在过去三年的创业经历中，是如何倚靠Python生态摸着石头过河，从无到有，逐步实现前后端分离、REST API、TDD、ATDD、自动部署等，追求用出Django、Flask等Web框架的最佳实践，同时通过实践检验了这些框架的优缺点。在这里与大家分享下，这一路走来，遇到了哪些坑，解决了哪些问题，最后总结Python语言在创业团队中应用的优劣。
茶歇&抽奖&合影
15:15-15:30
《Python代码道场》
15:30-17:30 张燎原
Python代码道场是让一群程序员聚在一起编程解决一些挑战；通过这种有趣的刻意的练习来提高技能。一边编程，一边互相学习。 每个人可以从Coding Dojo中学到：解决问题的思路、编程技巧、面向对象设计、演进式设计、结对编程、测试驱动开发、持续集成等等。其实即使从别人犯的错误中，我们也能够学到很多。
